# Configuração de Relatórios

Este diretório contém arquivos de configuração para relatórios.

## Arquivo de Configuração

Se você tiver um arquivo `report_config.json`, coloque-o aqui.

O caminho padrão é: `portal-backend/config/report_config.json`

Este arquivo é opcional se você não estiver usando Google Sheets para dados de relatórios.
