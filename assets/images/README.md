# Assets - Imagens para PDFs

Este diretório contém as imagens necessárias para geração de PDFs.

## Imagens Necessárias

Copie os seguintes arquivos de `repo/static/images/` para este diretório:

- `mr-consultoria-logo.png` - Logo da MR Consultoria
- `enel-logo.png` - Logo do cliente ENEL

## Como Copiar

```bash
# Da raiz do projeto
cp repo/static/images/mr-consultoria-logo.png portal-backend/assets/images/
cp repo/static/images/enel-logo.png portal-backend/assets/images/
```

Ou manualmente copie os arquivos para este diretório.
